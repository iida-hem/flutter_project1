## Brief instructions on how to use the application

The application main screen has a list of all entered recipes. The recipes are clickable and by pressing them the user can navigate to the recipe's page where its information is shown. The user can also interact with the recipes by liking them. On the Stats-page the user can see statistics on the recipes and their likes. On the Add recipe there is a form where the user can add a new recipe.

When running the app locally:
flutter run -t src/main.dart
Locally, the persistency of data may not work when using a browser to run the app.
